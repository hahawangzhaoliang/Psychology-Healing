## 🎯 开发流程

1. **Fork 仓库**
   ```bash
   git clone https://github.com/YOUR_USERNAME/Psychology-Healing.git
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **创建分支**
   ```bash
   git checkout -b feature/your-feature
   ```

4. **开发和测试**
   ```bash
   npm start
   # 在浏览器中测试功能
   ```

5. **提交代码**
   ```bash
   git add .
   git commit -m "feat: 添加新功能"
   git push origin feature/your-feature
   ```

6. **创建 Pull Request**
   - 描述您的更改
   - 关联相关 Issue
   - 等待审核
