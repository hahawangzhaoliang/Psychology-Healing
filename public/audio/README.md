# 音频文件说明

由于网络限制，音频文件需要手动下载。

## 手动下载步骤

### 1. 访问Pixabay Music
网址：https://pixabay.com/music/

### 2. 搜索并下载音频

#### 呼吸觉察（1分钟）
- 搜索：`breath of life 60 seconds`
- 下载后保存到：`public/audio/breathing/breathing_background.mp3`

#### 身体扫描（3分钟）
- 搜索：`meditation music calm relaxing`
- 下载后保存到：`public/audio/body-scan/body-scan_background.mp3`

#### 正念冥想（5分钟）
- 搜索：`meditation relaxing music`
- 下载后保存到：`public/audio/meditation/meditation_background.mp3`

#### 深度放松（7分钟）
- 搜索：`deep meditation`
- 下载后保存到：`public/audio/relaxation/relaxation_background.mp3`

### 3. 文件命名规范
```
[类型]_background.mp3

示例：
breathing_background.mp3
body-scan_background.mp3
meditation_background.mp3
relaxation_background.mp3
```
