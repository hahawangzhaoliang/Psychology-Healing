# 疗愈练习引导语音脚本

> 温暖、不施压、像朋友一样的引导语

---

## 一、呼吸觉察（1分钟）

### 女声版

**开场（5秒）**
```
现在，让我们花一分钟时间，关注你的呼吸。
```

**引导（40秒）**
```
找一个舒适的姿势坐下。
轻轻闭上眼睛，或者半睁半闭，都可以。

把注意力带到你的呼吸上。
不需要控制呼吸，只是观察它。

吸气时，感受空气进入鼻腔的清凉...
呼气时，感受空气离开身体的温热...

如果注意力跑开了，这很正常。
温和地把注意力带回到呼吸上就好。

就这样，简单地陪伴你的呼吸...
```

**结束（15秒）**
```
很好。
你刚刚给自己一分钟的时间，和自己的呼吸在一起。

现在，可以慢慢睁开眼睛。
带着这份平静，继续你的一天。
```

### 男声版

**开场（5秒）**
```
让我们用一分钟，回到当下。
```

**引导（40秒）**
```
找一个舒服的姿势。
闭上眼睛，或者微微垂下眼帘。

现在，把注意力放在你的呼吸上。
不需要改变什么，只是觉察。

吸气... 呼气...
吸气... 呼气...

如果思绪飘走了，没关系。
温柔地把它带回来，继续关注呼吸。

每一次呼吸，都是一次回到当下的机会。
```

**结束（15秒）**
```
做得很好。
这一分钟，你和自己在一起。

现在，可以睁开眼睛了。
感受一下身体的变化。
```

## 二、身体扫描（3分钟）

### 女声版

**开场（10秒）**
```
现在，让我们花三分钟时间，扫描你的身体。
找一个舒适的姿势，坐着或躺着都可以。
```

**引导（2分30秒）**

**双脚（20秒）**
```
首先，把注意力带到你的双脚。
感受脚底的触感，脚趾的温度。
不需要改变什么，只是觉察。
```

**小腿（20秒）**
```
现在，让注意力慢慢向上移动。
感受你的小腿...
肌肉的紧张或放松，温度的变化...
```

**大腿（20秒）**
```
继续向上，感受你的大腿...
如果你发现某个部位紧张，可以试着放松它。
或者，只是允许这种紧张存在。
```

**臀部和腹部（20秒）**
```
感受你的臀部接触椅子的感觉...
感受你的腹部，随着呼吸轻轻起伏...
```

**胸部（15秒）**
```
把注意力带到你的胸部...
感受心跳的节奏，呼吸的流动...
```

**双手和手臂（15秒）**
```
感受你的双手和手臂...
手指的温度，手掌的触感...
```

**肩膀和颈部（15秒）**
```
注意你的肩膀和颈部...
这里常常承载着很多压力。
试着让它们放松下来...
```

**面部（15秒）**
```
最后，感受你的面部...
放松眉头，放松下巴...
让整个面部变得柔和...
```

**全身（20秒）**
```
现在，感受整个身体作为一个整体...
从头到脚，完整地觉察...
身体在呼吸，身体在放松...
```

**结束（20秒）**
```
很好。
你刚刚用三分钟时间，完整地扫描了你的身体。

现在，可以慢慢睁开眼睛。
动动手指，动动脚趾。
带着这份身体的觉察，继续你的一天。
```

## 三、正念冥想（5分钟）

### 女声版

**开场（15秒）**
```
现在，让我们花五分钟时间，练习正念冥想。
找一个安静的地方，舒适的姿势。
```

**准备（20秒）**
```
轻轻闭上眼睛。
把手机调到静音，或者放在一边。
给自己这五分钟的时间，完全属于自己。
```

**呼吸觉察（1分钟）**
```
首先，把注意力带到你的呼吸上。
不需要控制呼吸，只是观察它。

吸气时，知道自己在吸气。
呼气时，知道自己在呼气。

呼吸是生命的节奏。
每一次呼吸，都是回到当下的锚点。
```

**身体放松（1分钟）**
```
现在，让身体放松下来。
从头顶开始，放松你的头皮...
放松你的面部...
放松你的肩膀...
放松你的手臂...
放松你的胸部和腹部...
放松你的背部...
放松你的双腿...

让整个身体变得柔软、放松。
```

**开放觉察（2分钟）**
```
现在，把觉察扩展到整个当下。

你可以觉察呼吸...
也可以觉察身体的感受...
或者觉察周围的声音...

不需要抓住任何体验。
也不需要推开任何体验。
只是允许一切如其所是地存在。

如果思绪飘走了，这很正常。
那是大脑在思考，这是它的工作。
温和地把注意力带回来就好。

就这样，简单地存在着。
不需要做任何事。
不需要成为任何人。
只是在这里，只是现在。
```

**结束（45秒）**
```
很好。
你刚刚给自己五分钟的时间，练习正念冥想。

现在，可以开始准备结束。
先动动手指，动动脚趾。
感受一下身体的存在。

然后，慢慢睁开眼睛。
看看周围的环境。
光线，颜色，形状...

带着这份平静和觉察，
继续你的一天。

记住，你可以随时回到呼吸，
随时回到当下。
```

## 四、深度放松（7分钟）

### 女声版

**开场（15秒）**
```
现在，让我们花七分钟时间，进入深度放松。
找一个舒适的地方，可以坐着或躺着。
```

**准备（20秒）**
```
轻轻闭上眼睛。
确保接下来的七分钟不会被打扰。
给自己这个时间，完全放松。
```

**渐进式肌肉放松（3分钟）**

**双脚和小腿（40秒）**
```
首先，注意你的双脚和小腿。
紧张你的脚趾，保持几秒...
然后放松，感受紧张和放松的区别。
紧张你的小腿肌肉...
然后放松，让紧张流走。
```

**大腿和臀部（40秒）**
```
现在，紧张你的大腿肌肉...
保持...然后放松。
感受大腿变得沉重、放松。
紧张你的臀部...
然后放松，让整个下半身放松下来。
```

**腹部和胸部（40秒）**
```
深吸一口气，紧张你的腹部...
然后呼气，放松。
感受腹部变得柔软。
紧张你的胸部...
然后放松，让呼吸自然流动。
```

**双手和手臂（40秒）**
```
握紧拳头，紧张你的双手和手臂...
保持...然后放松。
感受双手变得温暖、沉重。
让手臂完全放松。
```

**肩膀和颈部（40秒）**
```
耸起肩膀，紧张你的肩膀和颈部...
然后放松，让肩膀下沉。
感受紧张从颈部流走。
让整个上半身放松。
```

**面部（20秒）**
```

然后放松。
放松眉头，放松下巴，放松整个面部。
```

**引导式意象（3分钟）**

**场景构建（1分钟）**
```

可以是海滩，森林，或者任何让你感到平静的地方。

看看周围...
你看到了什么？
天空，树木，花朵...

听听周围的声音...
鸟鸣，风声，水流...

闻闻空气的味道...
清新的，自然的...
```

**沉浸体验（1分钟）**
```
感受阳光温暖地照在你身上。
微风轻拂你的皮肤。

你感到安全，平静，放松。
这里没有压力，没有烦恼。
只有宁静和美好。

在这个地方，你可以完全放松。
不需要做任何事，不需要担心任何事。
只是存在，只是享受。
```

**深度放松（1分钟）**
```
让这种放松的感觉渗透你的整个身体。
从头到脚，每一个细胞都在放松。

你的呼吸变得缓慢而深沉。
你的心跳变得平稳而有力。

你感到平静，安宁，完整。
```

**结束（45秒）**
```
很好。
你刚刚给自己七分钟的时间，进入深度放松。

现在，可以开始准备回到当下。
先动动手指，动动脚趾。
感受一下身体的存在。

然后，慢慢睁开眼睛。
看看周围的环境。
带着这份深度放松的感觉，
继续你的一天。

记住，你可以随时回到这个平静的地方。
```

## 五、录制建议

### 5.1 语音特点
- **语速**：慢速，约100字/分钟
- **音调**：柔和，平稳
- **停顿**：在句与句之间留出2-3秒停顿
- **语气**：温暖，不施压，像朋友一样

### 5.2 录制环境
- 安静的房间
- 使用高质量麦克风
- 避免背景噪音
- 保持一致的音量

### 5.3 后期处理
- 去除呼吸声和噪音
- 音量标准化
- 添加淡入淡出效果
- 与背景音乐混合（背景音乐音量20-30%）

### 5.4 文件格式
- 格式：MP3
- 比特率：128kbps或192kbps
- 采样率：44.1kHz
- 命名：`[类型]_guide_[性别].mp3`
