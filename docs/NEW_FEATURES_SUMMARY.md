**文件位置**：`/public/js/theme-manager.js`

  - 垃圾桶动画效果

- **娱乐主题** 🎮

## 使用指南

### 1. 集成主题管理器

在HTML页面底部添加：
```html
<script src="js/theme-manager.js"></script>
```

### 2. 应用主题样式

在CSS中使用CSS变量：
```css
body {
    background: var(--theme-background);
    color: var(--theme-text);
}

.primary-color {
    color: var(--theme-primary);
}
```

### 3. 页面跳转

```javascript
// 5分钟充电站
window.location.href = 'charging-station.html';

// 微光社区
window.location.href = 'community.html';

// 情绪记录
window.location.href = 'emotion.html';
```
