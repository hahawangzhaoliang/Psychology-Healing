## 🎨 设计亮点

### 1. 疗愈而非治疗
- 温暖、不施压的交互
- 像朋友一样的陪伴
- 不说教、不评判

### 2. 碎片化体验
- 1-7分钟快速练习
- 随时随地可使用
- 轻量级操作

### 3. 科学依据
- 基于积极心理学（PERMA模型）
- 正念冥想神经科学
- 情绪调节理论（Gross模型）

### 4. 情感化设计
- 三套主题皮肤
- 丰富的微动效
- 匿名保护隐私
- 个性化推荐
