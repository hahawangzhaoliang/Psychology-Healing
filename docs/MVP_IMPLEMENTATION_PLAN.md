# MVP实施计划

> 3个月快速验证核心价值

## 一、MVP核心目标

### 1.1 验证假设

**核心假设**：
1. 年轻人需要碎片化的心理疗愈工具
2. 3-10分钟的练习能产生实际效果
3. 身份认同视角能提升用户粘性
4. 科学背书能建立用户信任

### 1.2 成功指标

**用户指标**：
- 日活跃用户（DAU）：> 1,000
- 7日留存率：> 30%
- 30日留存率：> 15%

**行为指标**：
- 平均会话时长：> 5分钟
- 练习完成率：> 60%
- 情绪记录频率：> 3次/周

**满意度指标**：
- 用户满意度：> 4.0/5.0
- NPS（净推荐值）：> 30
- 应用商店评分：> 4.5

#### 功能1：碎片化疗愈练习

**用户故事**：
> 作为一个工作压力大的年轻人，我希望在工作间隙快速放松，这样我就能恢复精力继续工作。

**功能模块**：

**1.1 呼吸觉察（1分钟）**

**理论支撑**：
- 1分钟即可降低心率和皮质醇26%（2022年研究）
- 激活副交感神经系统，打破交感神经兴奋循环

**交互设计**：
```
页面布局：
┌─────────────────────────┐
│   [返回]  呼吸觉察      │
├─────────────────────────┤
│                         │
│      ◯ 呼吸动画         │
│    （圆圈扩张收缩）      │
│                         │
│   吸气... 呼气...       │
│                         │
│   [开始]  [暂停]        │
│                         │
│   剩余时间: 0:45        │
└─────────────────────────┘
```

**音频引导**：
- 温和的女声/男声（用户可选）
- 语速：慢速（约100字/分钟）
- 内容：
  - "现在，让我们花一分钟时间，关注你的呼吸..."
  - "吸气时，感受空气进入鼻腔的清凉..."
  - "呼气时，感受空气离开身体的温热..."
  - "如果注意力跑开了，没关系，温和地带回来..."

**视觉反馈**：
- 呼吸动画：圆圈随呼吸节奏扩张收缩
- 颜色：淡蓝色（平静）
- 进度条：底部显示剩余时间

**完成反馈**：
- "很好，你刚刚给自己一分钟的时间放松。"
- 显示心率变化（如果设备支持）
- 提供"再做一次"或"返回首页"选项

**技术实现**：
```javascript
// 呼吸动画控制
class BreathingAnimation {
  constructor() {
    this.inhaleTime = 4000;  // 吸气4秒
    this.holdTime = 2000;    // 屏息2秒
    this.exhaleTime = 6000;  // 呼气6秒
    this.totalDuration = 60000; // 总时长60秒
  }
  
  start() {
    // 开始呼吸循环动画
    this.breathCycle();
  }
  
  breathCycle() {
    // 吸气动画
    this.animateCircle('expand', this.inhaleTime);
    // 屏息
    setTimeout(() => {
      this.animateCircle('hold', this.holdTime);
    }, this.inhaleTime);
    // 呼气动画
    setTimeout(() => {
      this.animateCircle('contract', this.exhaleTime);
    }, this.inhaleTime + this.holdTime);
  }
}
```

// 计时器组件
class Timer {
  constructor(duration, onTick, onComplete) {
    this.duration = duration;
    this.remaining = duration;
    this.onTick = onTick;
    this.onComplete = onComplete;
  }
  
  start() {
    this.interval = setInterval(() => {
      this.remaining -= 1000;
      this.onTick(this.remaining);
      if (this.remaining <= 0) {
        this.stop();
        this.onComplete();
      }
    }, 1000);
  }
  
  stop() {
    clearInterval(this.interval);
  }
}

function preloadAudio(audioUrl) {
  const audio = new Audio();
  audio.preload = 'auto';
  audio.src = audioUrl;
  return audio;
}

// 音频缓存
const audioCache = new Map();

function getAudio(url) {
  if (!audioCache.has(url)) {
    audioCache.set(url, preloadAudio(url));
  }
  return audioCache.get(url);
}
