## 六、法律注意事项

### 6.1 Creative Commons许可类型

**CC0 (Public Domain)**：
- 完全免费
- 无需署名
- 可商用
- 可修改
- ✅ **最推荐**

**CC BY (Attribution)**：
- 免费
- 需署名
- 可商用
- 可修改
- ✅ **推荐**

**CC BY-SA (ShareAlike)**：
- 免费
- 需署名
- 可商用
- 修改后需相同许可
- ⚠️ **需注意**

**CC BY-NC (NonCommercial)**：
- 免费
- 需署名
- 不可商用
- ❌ **不推荐**

**CC BY-ND (NoDerivatives)**：
- 免费
- 需署名
- 可商用
- 不可修改
- ⚠️ **受限**

### 6.2 署名格式

如需署名，建议格式：
```
音乐来源：[作者名]
标题：[音频标题]
许可：Creative Commons Attribution [版本]
网址：[音频链接]
```

示例：
```
音乐来源：Grand_Project
标题：Breath of Life - 60 seconds
许可：Pixabay Content License
网址：https://pixabay.com/music/ambient-breath-of-life-60-seconds-320857/
```

### 6.3 商用定义

**属于商用**：
- 盈利性网站
- 付费应用
- 广告视频
- 商业演示

**不属于商用**：
- 个人博客
- 非营利组织
- 教育用途
- 公益项目

**心晴空间**：公益性质，但建议使用CC0或CC BY许可的音频
